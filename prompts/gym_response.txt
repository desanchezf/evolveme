NOTAS IMPORTANTES:
- "user": debe ser el username del usuario (string)
- "start_date" y "end_date": formato "DD/MM/YYYY HH:MM" (string)
- "warmup": texto opcional, puede ser null, string vacío o descripción del calentamiento
- "routine": objeto con días numerados (day-1, day-2, day-3, etc.)
- Cada día es un array de ejercicios
- Cada ejercicio debe tener:
  * "name": nombre del ejercicio (requerido, string)
  * "description": descripción opcional (puede ser null, string vacío o texto)
  * "body_part": parte del cuerpo, valores válidos: "chest", "back", "legs", "arms", "shoulders", "abs", "forearms", "core" (opcional, puede ser null o string vacío)
  * "sets": número de series (requerido, integer)
  * "reps": número de repeticiones (requerido, integer)
- Los campos opcionales pueden omitirse del JSON o enviarse como null/string vacío
