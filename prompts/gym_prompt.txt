# Rutina Push-Pull-Legs (PPL) Escalable

Rutina de **3-6 días/semana** para perder grasa preservando músculo.  
Nivel intermedio, sesiones de **60-90 min**.  
Calienta **5-10 min** bici/elíptica.  
**3-4 series x 8-12 reps**, **60-90s descanso**.  
Progresa peso/reps semanalmente.

---

## Estructura Semanal

- **3 días:** Lunes (**Push**), Miércoles (**Pull**), Viernes (**Legs**)
- **4 días:** Añade un día extra de Push o Pull
- **5 días:** Push – Pull – Legs – Push – Pull
- **6 días:** PPL x2 (con 1 día de descanso opcional)

---

## Día Push (Pecho, Hombros, Tríceps)

| Ejercicio                    | Series x Reps | Notas                      |
|------------------------------|---------------|----------------------------|
| Press banca/plano            | 4x8-10        | Controla bajada            |
| Press hombros mancuernas     | 3x10-12       | Sin arquear espalda        |
| Fondos pecho o máquina       | 3x10-12       | Añade peso si puedes       |
| Extensiones tríceps polea    | 3x12          | Mantén codos fijos         |
| **Core:** Plancha            | 3x30-45s      |                            |
| **Antebrazos:** Curl muñeca  | 3x15          | Opcional                   |

---

## Día Pull (Espalda, Bíceps)

| Ejercicio                        | Series x Reps | Notas                  |
|----------------------------------|---------------|------------------------|
| Dominadas o jalón pecho          | 4x8-10        | Aprieta escápulas      |
| Remo mancuerna/barra             | 3x10-12       | Tira con espalda       |
| Face pull hombros post.          | 3x12          | Para postura           |
| Curl bíceps barra                | 3x10-12       | Sin balanceo           |
| **Core:** Elev. piernas          | 3x12          |                        |
| **Antebrazos:** Giro muñeca inv. | 3x15          |                        |

---

## Día Legs (Piernas)

| Ejercicio                 | Series x Reps | Notas                 |
|---------------------------|---------------|-----------------------|
| Sentadilla/prensa         | 4x8-10        | Profundidad 90°       |
| Peso muerto rumano        | 3x10-12       | Protege lumbares      |
| Zancadas caminando        | 3x10/pierna   | Controladas           |
| Elev. gemelos             | 3x12-15       | Estira completo       |
| **Core:** Russian twists  | 3x20/lado     | Con peso ligero       |

---

## Progresión y Consejos

- Aumenta intensidad cada 1-2 semanas.
- Monitoriza fatiga en rutinas de 6 días.
- Nutrición: **2g proteína/kg**, déficit calórico moderado + ciclismo.
- Recupera con sueño; ajusta en caso de lesiones.
